The mass of one water molecule is approximately equal to 3x10 ^ -23 grams.
The mass of one drop of water is approximately 0.05g. In one faceted glass ≈249.5g.
Write a program that calculates the number of drops and water molecules in N faceted glasses of water.

Input data:
Natural number N - the number of glasses of water.

Output:
Two numbers. The first number is the number of drops in N glasses of water (integer).
The second number is the number of water molecules in N glasses of water, indicated with an accuracy of three decimal digits.

Масса одной капли воды приблизительно 0.05гр. В одном гранёном стакане ≈249.5гр.  
Напишите программу, которая вычисляет количество капель и молекул воды в N гранёных стаканах воды.

Входные данные:
Натуральное число N - количество стаканов воды.

Выходные данные:
Два числа.  Первое число - количество капель в N стаканах воды (целое число). 
Второе число - количество молекул воды в N стаканах воды, указанное с точностью до трёх десятичных цифр.



Sample Input 1:

1

Sample Output 1:

4990 8.317e+24

Sample Input 2:

3

Sample Output 2:
