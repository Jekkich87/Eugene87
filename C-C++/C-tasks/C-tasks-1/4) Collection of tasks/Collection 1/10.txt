Bitwise inversion
Write a program that inverts a single-bit number (0 or 1).

Input data:
The number K is either 0 or 1.

Output:
Print 1 if K = 0;
print 0 if K = 1;


Побитовая инверсия
Напишите программу, которая осуществляет инверсию однобитного числа (0 или 1).

Входные данные:
Число K либо 0, либо 1.

Выходные данные:
Вывести 1 , если K=0;
вывести 0, если K=1;
