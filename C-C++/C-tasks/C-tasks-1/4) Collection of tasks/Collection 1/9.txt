In modern society, there is confusion when using the words kilobytes, megabytes, etc.
In some cases, a kilobyte is understood as 10 ^ 3 = 1000 bytes, and in others 2 ^ 10 = 1024 bytes.
According to Wikipedia, decimal prefixes (degrees 10) are used for USB sticks.
Write a program that calculates how many bytes merchants deceive the consumer when they buy a USB flash drive on K - gigabytes.

Input data:
One natural number: K. 0 <K <32.

Output:
One number equal to the difference between the number of bytes in K binary gigabytes and K decimal gigabytes.


В современном обществе существует путаница при использовании слов килобайт, мегабайт и т.д.
В некоторых случаях под килобайтом понимается 10^3 = 1000 байт, а в других 2^10 = 1024 байт. 
Согласно Википедии для USB-флешек используются десятичные приставки (степени 10). 
Напишите программу, которая вычисляет на сколько байт коммерсанты обманывают потребителя при покупке им USB-флешки на K - гигабайт.

Входные данные:
Одно натуральное число: K. 0<K<32.

Выходные данные:
Одно число, равное разности между количеством байт в К двоичных гигабайтах и K десятичных гигабайтах.



Sample Input 1:

2

Sample Output 1:

147483648

Sample Input 2:

30

Sample Output 2:

2212254720
