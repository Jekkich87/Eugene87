From the beginning of the day k seconds have passed. Find the number of seconds elapsed since the start of the last minute.

Input data:
The integer k is the number of seconds elapsed since the beginning of the day.

Output:
An integer equal to the number of seconds elapsed since the beginning of the last minute.



С начала суток прошло k секунд. Найти количество секунд, прошедших с начала последней минуты.

Входные данные:
Целое число k - количество секунд, прошедших с начала суток.

Выходные данные:
Целое число, равное количеству секунд, прошедших с начала последней минуты.

Sample Input:

3292

Sample Output:

52
