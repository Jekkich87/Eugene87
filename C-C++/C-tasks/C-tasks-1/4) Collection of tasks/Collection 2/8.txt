The result of the operation "cyclic shift to the left" (by decimal place) for the integer 127 will be the number 271.
Write a program that performs the operation “cyclic left shift” for a three-digit integer k.

Input data:
A three-digit integer k.

Output:
The number obtained from k by cyclic left shift.

Входные данные:
Целое трёхзначное число k.

Выходные данные:
Число, полученное из числа k циклическим сдвигом влево.

Sample Input 1:

127

Sample Output 1:

271

Sample Input 2:

100
