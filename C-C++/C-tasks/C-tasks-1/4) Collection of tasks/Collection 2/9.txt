From the beginning of the day k seconds have passed. Find the number of seconds elapsed since the beginning of the last hour.

Input data:
The integer k is the number of seconds elapsed since the beginning of the day.

Output:
An integer equal to the number of seconds elapsed since the beginning of the last hour.



С начала суток прошло k секунд. Найти количество секунд прошедших с начала последнего часа.

Входные данные:
Целое число k - количество секунд прошедших с начала суток.

Выходные данные:
Целое число, равное количеству секунд прошедших с начала последнего часа.

Sample Input:

32792

Sample Output:

392
