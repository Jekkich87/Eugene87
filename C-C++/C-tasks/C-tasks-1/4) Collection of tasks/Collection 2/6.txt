The result of the left shift by decimal for the integer 127 is 1270.
Write a program that performs a left shift operation for a three-digit integer k.

Input data:
A three-digit integer k.

Output:
The number obtained from k by a left shift.



Результатом операция сдвиг влево на десятичный разряд для целого числа 127 будет число 1270. 
Написать программу, которая осуществляет операцию сдвиг влево для целого трёхзначного числа k.

Входные данные:
Целое трёхзначное число k.

Выходные данные:
Число, полученное из числа k сдвигом влево.



Sample Input:

505

Sample Output:

5050
