The result of the operation of cyclic shift to the right by a decimal place for the integer 127 will be the number 712.
Write a program that performs a cyclic shift operation to the right for a three-digit integer k.

Input data:
A three-digit integer k.

Output:
The number obtained from k by cyclic shift to the right.



Результатом операции циклическего сдвига вправо на десятичный разряд для целого числа 127 будет число 712. 
Написать программу, которая осуществляет операцию циклического сдвига вправо для целого трёхзначного числа k.

Входные данные:
Целое трёхзначное число k.

Выходные данные:
Число, полученное из числа k циклическим сдвигом вправо.



Sample Input 1:

127

Sample Output 1:

712

Sample Input 2:

100

Sample Output 2:

10
