There is k day of the year. Determine the number of the current week in the year. We assume that January 1 fell on Friday.

Input data:
The integer k, 1≤k≤365.

Output:
A single integer is the week number.



Идёт k день года. Определить номер текущей недели в году. Предполагаем, что 1 января выпало на пятницу.

Входные данные:
Целое число k , 1≤k≤365.

Выходные данные:
Одно целое число - номер недели. 



Sample Input 1:

3

Sample Output 1:

1

Sample Input 2:

12

Sample Output 2:

3

Sample Input 3:

364

Sample Output 3:

53

Sample Input 4:

11

Sample Output 4:

3
