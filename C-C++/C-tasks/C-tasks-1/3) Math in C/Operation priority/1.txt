Write a program that calculates the area of a triangle on three sides.

Input Format:

Three numbers a, b, c — the lengths of the sides of the triangle — enter the program input.
a, b, c are real numbers.

Output Format:
One number is the area of the triangle with sides a, b, c. Accuracy - two decimal places.

Hint: To calculate the area, use the Heron formula.


Написать программу, вычисляющую площадь треугольника по трём сторонам.

Формат входных данных:

На вход программы поступают три числа a,b,c — длины сторон треугольника.
a,b,c — вещественные числа.

Формат выходных данных:
Одно число — площадь треугольника со сторонами a,b,c. Точность — два знака после запятой.

Подсказка: Для вычисления площади используйте формулу Герона. 



Sample Input:

12.19 12.11 17.18

Sample Output:

73.81
