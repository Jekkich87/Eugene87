The number of integers that can be written using N binary bits is calculated by the formula 2 ^ N.

Write a program that calculates the number of integers that can be written using N bits.

Input Format:
The input of the program receives one integer N - the number of binary bits.

Output Format:
One integer is the number of integers that can be written using N bits.


Количество целых чисел, которые можно записать используя Nдвоичных битов вычисляется по формуле 2^N .  

Напишите программу, вычисляющую количество целых чисел, которые можно записать используя N битов.

Формат входных данных: 
На вход программы поступает одно целое число N— количество двоичных битов. 

Формат выходных данных:
Одно целое число — количество целых чисел, которые можно записать, используя N битов.



Sample Input:

11

Sample Output:

2048
