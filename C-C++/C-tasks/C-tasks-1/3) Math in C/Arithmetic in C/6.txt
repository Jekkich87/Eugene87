Write a program that defines even numbers.

The input of the program receives a single non-negative integer.

The program should print 1 if the number is even, and 0 if the number is odd.



Напишите программу, определяющую чётные числа.

На вход программы поступает одно целое неотрицательное число. 

Программа должна вывести 1, если число чётное, и 0, если число нечётное.



Sample Input:

2

Sample Output:

1
